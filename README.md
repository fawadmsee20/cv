This repository hosts the personal academic website of **Fawad**, PostDoc Researcher at EnergyAI Lab, KENTECH, South Korea.

## 🌐 Live Site
**https://YOUR-GITHUB-USERNAME.github.io/**  
*(Replace `YOUR-GITHUB-USERNAME` with your actual GitHub username)*

## 📂 File Structure
```
index.html          ← Homepage
research.html       ← Research interests
publications.html   ← Publications list
experience.html     ← Professional experience & education
education.html      ← Education, awards, skills
cv.html             ← CV page with PDF viewer
style.css           ← Design and styling
app.js              ← Interactivity (navigation, mobile menu)
CV_Fawad.pdf        ← Your CV PDF (upload this!)
```

## 📄 Adding Your CV PDF
1. Rename your CV PDF file to `CV_Fawad.pdf`
2. Place it in the same folder as `index.html`
3. Commit and push – it will automatically appear on the CV page
